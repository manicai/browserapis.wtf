Text Formatting

Please note these are all inline elements, so please use them in sentences (<p>). The excepting is the anchor tag, which officially can be a block element in HTML5

Spans are cool, you can just wrap em around anything inline and style willy nilly

<span>Add a class for styling</span>

Featured text. For that extra special copy. Add a class of feature-text to a span to make text bold and italic.

<span class="feature-text">Featured text</span>

This sentence uses the strong tag.

<strong>This is important</strong>

This sentence uses the b tag, it styles like strong but has no semantic meaning.

<b>I'm a fake strong</b>

This sentence uses the emphasis tag.

<em>This is emphasised</em>

This sentence uses the i tag, it styles text italic like.

<i>I'm italic style text</i>

You'll notice now I'm going to.
Break this sentence with a br. These should not be used to clear elements.

<br />

This is an inline link. Note the rel atrribute :)

<a href="#" rel="external">I'm a link</a>

This is the keep del element

<del>Delete this</del>

I want to be highlighted

<mark>highlight me</mark>

I'm going to insert something here.

<ins>inserted</ins>

This is defined, hover it.

<dfn title="definition here">define</dfn>

The UK is abbreviated

<abbr title="long version here"></abbr>

If you're writing code you may use this.

<code>code here</code>

So there's a keyboard tag. Press Enter

<kbd>Enter</kbd>

There's also sample output from a program.

<samp>sample output</samp>

A simple equation: x = y + 2

<var>variable here</var>

This is tiny text

<small>small text</small>

The sub text element

<sub>sub text</sub>

The sup text element

<sup>sup text</sup>

Code

Wrap any code in a pre element and a code element

<!DOCTYPE html>
<html>
<head>
	<title>An example of HTML code</title>
</head>
<body>
	<!-- CONTENT HERE -->
</body>
</html>
<pre class="language-markup"><code><!DOCTYPE html>
<html>
<head>
	<title>An example of HTML code</title>
</head>
<body>
	<!-- CONTENT HERE -->
</body>
</html></code></pre>

Notice the class on the pre tag. You can define any language here, like Ruby for instance

def ask_yoda(question)
	yoda_array = ['do', 'do not']
	message = yoda_array.join(' or ') + ' there is no ' + question
	return message unless yoda_array.include?(question)
	'you are strong with the force'
end
