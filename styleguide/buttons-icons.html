Buttons & Icons

Buttons

Lots of elements can easily take on the button style, either by the use of the correct element (ಠ_ಠ) or by adding a class of .button. Check out _buttons-icons.scss for all the styles.

<button>Just a button</button>
<input type="submit" value="valueHere" name="nameHere" />
Anchor 
<a href="/" class="button">Anchor</a>

You can easily differ the colour, if the button is unique, by calling the button-style mixin and passing in a colour.

.button-reset, input[type="reset"] {@include button-style($pink)}

Form resets differ by default.

<input type="reset" value="valueHere" name="nameHere" />

The disabled attribute can be added to all inputs and buttons, or you can use the .button-disabled class. 

<button disabled>button:disabled</button>

If you want something a little more eye catching, try adding the .button-feature class (use it with .button class on any element).

<button class="button-feature">Main Call To Action</button>

Icons

We use an icon font called Font Awesome this is because of the huge amount of icons that come with it, one for every occasion.

There are four ways to include icons: Turn as element into an icon, turn an element into a button with an icon, add an icon at the start of an element and add an icon at the end of an element.

Each way uses it's appropriate class, then a class to depict which icon to use, (see bottom reference), like i-refresh or i-menu. Any item can be used with any method.

You can also use sass' @extend directorate to add the styles from these classes on to your element in the CSS, keeping the HTML clean. Hurrah \o/

Turn an element into an icon: .icon-make

Use appropriate button or link elements if the icon is clickable.

<button class="icon-make i-close">Close</button>
Refresh 
<a href="" class="icon-make i-refresh">Refresh</a>

A span is always good if it's not, however any element can be used.

Settings 
<span class="icon-make i-settings">Settings</span>

Turn an element into a button with an icon: .icon-button

N.B. You can't use these icon methods on input elements, for form buttons please use the button element instead.

<button class="icon-button i-feedback">Feedback</button>
Email 
<a href="" class="icon-button i-email">Email</a>

Add an icon at the start of an element: .icon-before

<button class="icon-before i-camera">Take Photo</button>
Account 
<a href="" class="icon-before i-user">Account</a>
Completed 
<span class="icon-before i-yex">Completed</span>

Add an icon at the end of an element: .icon-after

<button class="icon-after i-calendar">Show Calendar</button>
Sign Out 
<a href="" class="icon-after i-signout">Sign Out</a>
Tweet 
<span class="icon-after i-twitter">Tweet</span>
