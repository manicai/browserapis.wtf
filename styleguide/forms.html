Forms

Input types

Check out _forms.scss for all the styles. If you want a quick complete form check out the example forms at the bottom of this page

Before you dive right in copying and pasting the inputs you want, don't forget you need a form! (Don't worry the legend shouldn't be displayed as default, but is always good to have for accessibility).

This is a form! 
<form method="post" action="/">
  <fieldset><legend>This is a form!</legend>
    <!--Form inputs here-->
  </fieldset>
</form>

You may want instructions or hints throughout the form, just drop in a paragraph with the class .form-instructions.

This is a form!

Some instructions or hints here: Lorizzle ipsum dolor mofo amet, consectetuer adipiscing fo shizzle my nizzle.

<form method="post" action="/">
  <fieldset><legend>This is a form!</legend>

    <!--Form inputs can be here-->
  </fieldset>
</form>

  <label for="text-input">Text input</label>

</div>

If a field needs to be required, just add the .input-required class to the wrapping div. (You'll see this automagically adds an asterix to the label).

Errors And Successes

To add success or error styling to inputs just add .input-success or .input-error classes to the wrapping div. 

<div class="form-field input-success">
  <label for="text-input">Text input</label>
  <input type="text" name="text-input" id="text-input" placeholder="Some text" />
</div>
<div class="form-field input-error">
  <label for="text-input">Text input</label>
  <input type="text" name="text-input" id="text-input" placeholder="Some text" />
</div>

For a form success or error message just create a paragraph with the relevant class:

You're an idiot, you can't fill out a form correctly. Try again *sigh*

<p class="form-error">You're an idiot, you can't fill out a form correctly. Try again *sigh*</p>

You're brilliant! The form was filled out perfectly :)

<p class="form-success">You're brilliant! The form was filled out perfectly :)</p>

Example forms

The default form layout is full width form. For smaller forms try a half width form (see below).

Register 
<form method="post" action="/">
  <fieldset><legend>Register</legend>
    <div class="form-field">
      <label for="full-name">Full Name</label>
      <input type="text" name="full-name" id="full-name" placeholder="Joe Bloggs" />
    </div>
    <div class="form-field">
      <label for="email-input">Email input</label>
      <input type="email" name="email-input" id="email-input" placeholder="example@domain.com" />
    </div>
    <div class="form-field">
      <label for="password-input">Password</label>
      <input type="password" name="password-input" id="password-input" />
    </div>
    <div class="form-field">
      <label for="reenter-password">Re-enter password</label>
      <input type="password" name="reenter-password" id="reenter-password" />
    </div>
    <input type="submit" value="Register" />
  </fieldset>
</form>

Add the class.form-small for forms with less fields.
