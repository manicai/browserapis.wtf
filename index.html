Browser APIs

AKA Web APIs, Web Javascript APIs, Device APIs, HTML5 APIs (meh), or simply Javascript APIs

I like Client Side Web APIs myself - or CSWAPIS if you will

  • Ambient Light Event

    Returns the lux value detected by the devices camera.

  • Web Animation API

    Specifying and controlling DOM animations.

  • Web Audio API

    Allows manipulation and analysation of audio data (files or streams), as well as play back controls.

  • Device Orientation API

    Returns values on the device's position & orientation.

  • Geolocation API

    User grants access their to device location, accessible via lat/long co-ords.

  • Media Stream API

    User grants access to stream device media, usually either camera and/or microphone.

  • Vibration API

    Causes the device to vibrate.
